# Spring IA Frontend React

React frontend integrated with a Java 21 backend using Spring Boot and Spring AI to consume OpenAI APIs and interact with ChatGPT.

## 🚀 About the Project

This project is a React frontend application responsible for consuming APIs exposed by a Java 21 backend built with Spring Boot and Spring AI.

The application allows integration with OpenAI models and includes features such as:

* Chat with ChatGPT
* REST API consumption
* Upload and send data for AI processing
* Smart responses using Spring AI
* Modern integration between React and Spring Boot

The main goal of this project is to provide a practical Full Stack example using:

* React
* Axios
* Java 21
* Spring Boot
* Spring AI
* OpenAI API

## 🛠️ Technologies Used
